<h1>Installation</h1>
** Requirements
```bash
apt-get install xorg i3 compton scrot xcompmgr vim feh 
```
** Clone repository into ~/opt
```bash
git clone https://github.com/kimbos/dotfiles.git ~/opt
```
** Run the install script
```bash
~/opt/dotfiles/install.sh
```
